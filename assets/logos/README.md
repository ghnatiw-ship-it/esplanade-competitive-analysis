Place brand and venue logo files in this folder.

Recommended filenames:

- `esplanade_restaurants.png`
- `bar_cart.png`
- `bar_cathedral.png`
- `eloise.png`
- `scotland_yard.png`
- `old_spaghetti_factory.png`

Supported formats:

- `.png`
- `.jpg`
- `.jpeg`
- `.svg`
- `.webp`

The app auto-detects logos by filename match and displays them in the main header and venue pages.
