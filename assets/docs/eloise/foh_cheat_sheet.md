# Eloise — FOH Cheat Sheet For Taste Eloise Signature

> Purpose: help hosts and managers explain the peak-time experience clearly and confidently

## What It Is

`Taste Eloise Signature` is a curated four-course dinner available on selected prime dinner seatings through OpenTable.

It is not:

- a surcharge
- a discount bundle
- a secret minimum spend rule

It is a premium reservation type built from Eloise's existing menu.

## When It Applies

- Thursday to Saturday
- `6:30 p.m.`
- `7:00 p.m.`
- `7:30 p.m.`

Standard dinner reservations remain available at other times.

## Core Talking Points

- It is a curated four-course dinner built from Eloise's signature dishes.
- It is designed for our prime dinner seating.
- It offers a more complete Eloise experience.
- The whole table participates.
- Optional wine pairing is available.

## Host Script

If a guest asks why only this option is available:

`For those prime dinner seatings we offer Taste Eloise Signature, which is our curated four-course dinner experience. If you prefer standard a la carte dining, we still offer that at our earlier and later reservation times.`

If a guest asks whether it is a minimum spend:

`No, it is not a minimum-spend charge. It is a curated dinner format built from our menu and offered as its own reservation type during select prime-time seatings.`

If a guest asks whether the menu is discounted:

`It is not a discounted menu. It is designed as a complete Eloise dinner using our signature dishes.`

If a guest asks whether everyone has to participate:

`Yes, the experience is served for the whole table.`

If a guest wants a la carte at `7:00 p.m.`:

`At that time we are offering the Taste Eloise Signature experience. If you would like to dine a la carte, I can help you with the closest available earlier or later seating.`

## Escalation Rules

Manager should step in if:

- the guest is a VIP or known regular
- there is a serious allergy or dietary complexity
- the guest is considering cancelling altogether
- the party is large enough to justify a custom accommodation

## Good Language

- curated
- signature
- experience
- prime dinner seating
- complete evening
- whole-table participation

## Bad Language

- minimum spend
- mandatory spend
- premium fee
- surcharge
- forced menu

## Success Metrics To Track

- conversion rate by prime-time slot
- average cheque for experience covers
- number of guests redirected to a la carte later seatings
- objections or confusion logged by hosts
- review comments related to the experience
